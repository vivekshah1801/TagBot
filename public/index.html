TagBot.tech

Tags people by recognizing their faces in images sent in a Discord server.

+ Add To Your Server
  • Uses Robost Python face-recognition library
  • Uses Firebase's Firestore as mapping Database
  • Hosted on Fast and Reliable Google-Cloud Infstracture

Inspiration

Nowadays most of the people share images of their friend in social media group.But it not tag the people present in image and group so we thought that it would be more fun if there is something which tag the people who is in images.

What It Does ?

It takes user image from discord as input and trained it.And when someone share some image on discord then if user is present in image then it will tag that user.

How we built it ?

We have two services one bot service takes image from user and another face - recognition service detect the faces and return the userid of detected faces who is present in server.
